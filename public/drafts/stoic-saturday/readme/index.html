Stoic Saturday Drafts#

Place future Stoic Saturday posts here.

Naming: stoic-saturday-N.md where N is the issue number.

When: Every Friday at noon CDT, the workflow checks for drafts. What: The workflow moves the next draft to content/posts/, updates the date to the upcoming Saturday, and triggers the site rebuild.

Don’t commit a post to content/posts/ manually if you want the workflow to handle it.
